1. ОПИСАНИЕ ПРОЕКТА
1.1 Цель проекта
Разработка комплексной информационной системы управления образовательным процессом для Urban College (г. Астана, Казахстан).

1.2 Основные функции
Модуль столовой - учет питания студентов, QR-коды, лимиты, категории
Управление пользователями - CRUD для студентов, преподавателей, операторов
Управление группами - создание групп, назначение студентов
Управление предметами - предметы, назначение преподавателей
Система оценок - выставление и просмотр оценок
Посещаемость - отметка через одноразовые токены
Расписание - управление занятиями
Личный кабинет студента - QR-код, баланс, оценки
1.3 Технологический стек
Backend:

PHP 7.4+
PDO (MySQL)
RESTful API
JWT Authentication
Frontend:

HTML5, CSS3, JavaScript (Vanilla)
Mobile-First responsive design
QR Code Scanner (html5-qrcode)
База данных:

MySQL 8.0
InnoDB Engine
UTF8MB4 Collation
