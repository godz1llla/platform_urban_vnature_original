# Quick Start: URBAN COLLEGE PLATFORM

## Этапы 1-4 ЗАВЕРШЕНЫ ✅

Платформа готова к запуску с полной системой управления:
- ✅ Пользователи (студенты, преподаватели, операторы)
- ✅ Группы
- ✅ Предметы
- ✅ Расписание

---

## Что нужно сделать

### 1. Интеграция HTML (ОБЯЗАТЕЛЬНО)

Следуйте инструкции в файле:
```
INTEGRATION_GUIDE.md
```

Вставить 8 HTML файлов в index.html:
- 4 секции (users, groups, subjects, schedule)
- 4 модалки (формы создания/редактирования)

### 2. Загрузить файлы на хостинг

**Backend:**
- database/full_database.sql (импортировать в MySQL)
- backend/ (все файлы)

**Frontend:**
- frontend/ (все файлы)

### 3. Настроить .env

Создать файл `.env` в корне backend/:
```env
DB_HOST=localhost
DB_NAME=urban_college
DB_USER=root
DB_PASS=your_password
SECRET_KEY=your_jwt_secret_key_here
```

### 4. Первый запуск

1. Импортировать `database/full_database.sql`
2. Создать первого админа вручную в БД:
   ```sql
   INSERT INTO users (full_name, email, password, role)
   VALUES ('Admin', 'admin@urbancollege.kz', '$2y$10$...', 'admin');
   ```
   (хеш пароля через bcrypt)

3. Логин: admin@urbancollege.kz

## Использование

### Админ:
1. Создать пользователей (студенты, преподаватели)
2. Создать группы
3. Добавить студентов в группы
4. Создать предметы
5. Назначить преподавателей на предметы
6. Создать расписание

### Студент:
- Видит своё расписание
- Может получать питание в столовой (QR)

### Преподаватель:
- Видит свои занятия
- (В будущем: журнал, ДЗ)
